# 个人简历网站

## 🚀 项目介绍
这是一个基于 **Vue 3 + TypeScript + Vite** 开发的个人简历网站，使用 **TailwindCSS** 进行样式设计，并支持**深色模式**切换。

## 🎨 主要功能
- 个人介绍
- 技能展示
- 项目展示
- 联系方式
- **支持深色模式切换**

## 🛠️ 技术栈
- **前端框架**：Vue 3
- **编程语言**：TypeScript
- **状态管理**：Pinia
- **构建工具**：Vite
- **样式库**：TailwindCSS + Heroicons
